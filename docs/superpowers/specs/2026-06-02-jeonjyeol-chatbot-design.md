# 전결 자연어 챗봇 — 설계문서

작성일: 2026-06-02
상태: 승인됨 (구현계획 전)

## 1. 목적
동구 위임전결규정을 자연어로 물어 **"내가 ○○ 업무할 때 누구한테 전결받나"** 를 답하는 웹 챗봇.
데이터는 별도 프로젝트에서 만든 `전결_검색테이블_통합.csv`(169행)를 사용한다.

## 2. 핵심 요구사항 (확정)
- **LLM 기반** 자연어 이해. **Claude(Anthropic)/OpenAI 둘 다 지원**, `LLM_PROVIDER` 환경변수로 전환.
- **멀티턴 대화**: 분기기준(금액/직급/중요도/부서수/기간)에 따라 되물은 뒤 전결권자 안내.
- **클라우드 배포(Vercel) + GitHub push 자동배포**. 동료가 공개 URL로 접속.
- **인증 없음(공개 v1)**.
- 데이터(169행)가 작아 **표 전체를 시스템 프롬프트에 주입 + 프롬프트 캐싱**(벡터DB/RAG 불필요).

## 3. 비목표 (YAGNI)
- 로그인/계정, 대화 영구저장, 관리자 화면, 다국어, 표 편집 UI — v1 제외.
- RAG/임베딩 — 169행엔 과잉.

## 4. 아키텍처
```
브라우저 (React 챗 UI)
   ↕ POST /api/chat   (대화 히스토리 전달, 무상태)
Next.js 서버 (Vercel) ── API 키 보관, 시스템 프롬프트 조립
   ↕ LLM API (Claude | OpenAI, env 전환)
데이터: data/전결_검색테이블_통합.csv → 시스템 프롬프트에 주입(캐싱)
```

## 5. 구성요소 (각 1책임, 독립 테스트 가능)
- `app/page.tsx` — 챗 UI. 메시지 목록 + 입력창. 봇 되물음을 그대로 렌더. (금액/직급 선택 버튼은 v1.1 후보)
- `app/api/chat/route.ts` — POST 핸들러. 대화 히스토리 받아 `callLLM` 호출, assistant 메시지 반환. 무상태.
- `lib/llm.ts` — `callLLM(provider, system, messages)`. Anthropic/OpenAI 추상화. 시스템 프롬프트 캐싱.
- `lib/table.ts` — CSV 로드 → 프롬프트용 간결 텍스트(또는 JSON) 변환. 서버 시작 시 1회.
- `lib/systemPrompt.ts` — 역할·표·분기규칙·출력형식·근거규칙 조립.
- `data/전결_검색테이블_통합.csv` — 데이터(원본 그대로).

## 6. 데이터 흐름
1. 사용자: "출장보고 누구 전결?"
2. 프론트가 전체 메시지 히스토리를 `/api/chat`에 POST
3. 서버: `[system(표+규칙, 캐시), …history, user]` 구성
4. LLM: 충분하면 전결권자 답변, 부족하면 분기 되물음("중요사안인가요, 경미한가요?")
5. 프론트 렌더 → 사용자 응답 → 반복

## 7. 시스템 프롬프트 — 근거 규칙 (감사 안전 핵심)
- 표에 있는 내용만 답한다. 없으면 "원문에 규정 없음"이라 답하고 **전결권자를 지어내지 않는다**.
- 금액 질문은 `금액하한/금액상한` **구간**으로 판단한다(문자열 "이하" 직역 금지).
- 분기기준이 '없음'이 아니면 부족한 정보를 **먼저 되묻는다**(금액/직급/중요도/부서수/기간).
- 비고의 **준용·gap·단독전결**을 답에 노출한다(예: 관내출장="나목 준용", 상한초과="원문 미규정").
- 직급분기는 비고의 '직급대상'(본인/제3자: 표창 대상자, 회의 참석자 등)을 반영해 되묻는다.
- 출력형식: "○○님, **△△께 결재(전결) 받으시면 됩니다**" + 근거(대분류·세부·조건) 한 줄.
- 화면 하단 상시 고지: "위임전결규정 기반 참고용입니다. 최종 확인은 원규정/담당부서."

## 8. 오류 처리
- LLM API 오류 → 사용자에게 재시도 안내, 서버 로그.
- env 누락(`LLM_PROVIDER`/키) → 명확한 5xx + 설명.
- 빈 입력/과도한 길이 → 기본 검증.

## 9. 테스트 (골든 회귀 케이스)
`test/golden.test.ts` — 어댑터/프롬프트가 아래를 만족하는지 확인(LLM 응답 또는 규칙엔진 레벨):
| 질의 | 기대 전결 |
|---|---|
| 출장보고(경미) | 실·단·과장 |
| 출장보고(중요/과장이상 결과보고) | 부구청장 |
| 3천만원 공사 집행(추정금액) | 국·소장 (구간 2천만초과~1억) |
| 병가 1일이상, 6급 이하 | 실·단·과장 |
| 관내출장, 일반직원 | 팀장 (나목 준용) |
| 예산의 변경 | 국·소장 |
