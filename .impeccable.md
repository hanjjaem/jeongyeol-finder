## Design Context

### Users
부산 동구청 공무원. 결재(기안)를 올리기 직전, 책상 PC 또는 휴대폰에서 "이 업무는 누구에게 전결받아야 하나"를 빠르게 확인한다. 행정 업무 맥락이라 **신속·정확**이 최우선이고, 답(전결권자)이 한눈에 보여야 한다.

### Brand Personality
신뢰감 있고 효율적인 생산성 도구. 3단어: **정확한 · 깔끔한 · 빠른**. 사용자가 "믿고, 빠르게, 정확한 답을 얻는" 감각을 받아야 한다. 과장된 화려함보다 절제된 세련됨.

### Aesthetic Direction
**모던 SaaS·테크형** — 깔끔한 현대 생산성 앱. **라이트/다크 토글** 지원(둘 다 동등 품질). 강조색은 **정제된 블루**(현재 #2563eb 계열을 더 다듬어 사용). 부드럽되 과하지 않은 그림자·라운드.
- 회피(안티레퍼런스): 글래스모피즘 남발, 네온/글로우, 보라-블루 그라데이션, 회색 텍스트 위 컬러배경, 순수 흑/백, AI슬롭 카드 격자.

### Design Principles
1. **신뢰 우선** — 전결권자(=답)가 즉시 눈에 띄게, 정보 위계를 분명히.
2. **정제된 모던** — 표면·여백에 리듬, 장식은 목적이 있을 때만.
3. **듀얼 테마 토큰화** — CSS 변수로 라이트/다크를 같은 품질로, 중립색은 블루로 살짝 틴트.
4. **빠른 체감** — 즉각 반응·부드러운 모션(transform/opacity), 낙관적 UI.
5. **모바일 동등** — 폰에서도 풀 기능·가독성(현재 주요 사용처가 모바일).
